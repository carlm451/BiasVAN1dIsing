# Figures

All figures are generated by scripts in `figures/` and saved to `figures/output/` as both PDF and PNG.

## Dependencies

Each figure script requires specific result files. The table below shows which experiments must run first.

| Figure | Script | Requires | Can run standalone? |
|--------|--------|----------|-------------------|
| Fig 2 | `plot_exact_phase_diagram.py` | `sweep_Kh_N16.npz` or nothing | Yes (computes from transfer matrix if no data) |
| Fig 3 | `plot_delta_F_heatmap.py` | `sweep_Kh_N16.npz` | No |
| Fig 4 | `plot_bias_ablation_heatmap.py` | `sweep_Kh_N16.npz` | No |
| Fig 5 | `plot_H0_slice.py` | `sweep_Kh_N*.npz` (any) | No |
| Fig 6 | `plot_magnetization.py` | `sweep_Kh_N16.npz` | No |
| Fig 7 | `plot_finite_size.py` | `finite_size.npz` | No |
| Fig 8 | `plot_n2_verification.py` | `n2_verification.npz` | No |
| Fig 9 | `plot_parameters.py` | Nothing (trains live) | Yes |
| Fig 10 | `plot_convergence.py` | Nothing (trains live) | Yes |

## Figure Descriptions

### Fig 2: Exact Phase Diagram
**File:** `fig2_exact_phase_diagram.{pdf,png}`

Heatmap of exact magnetization $\langle m \rangle$ in the $(K, h)$ plane. Color map: RdBu_r (red = $m > 0$, blue = $m < 0$). Y-axis (K) is log-scaled.

Shows the crossover from disordered ($m \approx 0$ at small $K$) to field-aligned ($|m| \approx 1$ at large $K$ or large $|h|$). No true phase transition in 1D, but a sharp crossover at $h = 0$ for large $K$.

### Fig 3: Free Energy Error Heatmap
**File:** `fig3_delta_F_heatmap.{pdf,png}`

Side-by-side heatmaps comparing $\Delta F = \beta f_\text{approx} - \beta f_\text{exact}$:
- **Left panel:** NMF error
- **Right panel:** VAN (with bias) error

Color map: hot_r (darker = larger error). Both panels share the same color scale. Y-axis (K) is log-scaled.

Key observation: VAN error is dramatically smaller than NMF everywhere, especially at large $K$ where correlations are strong.

### Fig 4: Bias Ablation Heatmap
**File:** `fig4_bias_ablation_heatmap.{pdf,png}`

Side-by-side heatmaps of VAN error:
- **Left panel:** VAN without bias
- **Right panel:** VAN with bias

Demonstrates that bias terms are essential when $h \neq 0$. At $h = 0$ both versions perform similarly.

### Fig 5: h=0 Slice
**File:** `fig5_H0_slice.{pdf,png}`

$|\Delta F / N|$ vs $K$ at $h = 0$ for multiple system sizes. Semi-log plot. Auto-detects available sweep files.

Shows how VAN error depends on coupling strength and system size along the zero-field axis.

### Fig 6: Magnetization vs h
**File:** `fig6_magnetization.{pdf,png}`

Magnetization $\langle m \rangle$ vs $h$ at fixed $K = 2.0$, comparing exact and NMF for N=16.

Shows the magnetization curve (tanh-like) and how NMF differs from the exact solution.

### Fig 7: Finite-Size Scaling
**File:** `fig7_finite_size.{pdf,png}`

Log-log plot of $\Delta F / N$ vs $1/N$ at representative $(K, h)$ points. Error bars from multiple seeds. Reference line shows $\propto 1/N$ scaling.

Confirms the VAN error arises from the autoregressive ordering artifact on the periodic wrap-around bond, which contributes $O(1/N)$ error per spin.

### Fig 8: N=2 Verification
**File:** `fig8_n2_verification.{pdf,png}`

Heatmap of $\log_{10}|\Delta F|$ for N=2 across the $(K, h)$ grid. Color map: viridis. Y-axis (K) is log-scaled.

Expected to be uniformly very small ($< -4$ everywhere), confirming that the 3-parameter VAN exactly represents the 4-state N=2 Ising model.

### Fig 9: Converged Parameters
**File:** `fig9_parameters.{pdf,png}`

Two panels showing converged VAN parameters at N=16, K=1.0 for several field values:
- **Left:** Bias $b_i$ vs site index
- **Right:** Nearest-neighbor weight $W_{i,i-1}$ vs site index

Shows translational-invariance breaking from the autoregressive ordering (parameters depend on position in the chain).

### Fig 10: Convergence Curves
**File:** `fig10_convergence.{pdf,png}`

$\beta F_q / N$ vs training step for representative $(K, h)$ points at N=16. Horizontal dashed lines show exact $\beta f$.

Demonstrates training convergence behavior: high-T (small K) converges fast, low-T (large K) converges slower.

## Generating Figures

### After running experiments

```bash
source venv/bin/activate

# Individual figures
python figures/plot_exact_phase_diagram.py
python figures/plot_delta_F_heatmap.py
python figures/plot_n2_verification.py
# ... etc

# Or all at once (via run_all.py after experiments complete)
python experiments/run_all.py
```

### Standalone figures (no experiment data needed)

```bash
source venv/bin/activate
python figures/plot_exact_phase_diagram.py  # computes from transfer matrix
python figures/plot_parameters.py           # trains VAN live
python figures/plot_convergence.py          # trains VAN live
```

### Customizing

Most plot functions accept keyword arguments:

```python
from figures.plot_magnetization import plot
plot(N=8, K_target=1.0)  # Different system size and coupling

from figures.plot_convergence import plot
plot(N=8, test_points=[(0.5, 0.0, 'K=0.5'), (3.0, 1.0, 'K=3, h=1')])
```
