# Results File Format

All results are saved as NumPy `.npz` files in the `results/` directory. Load them with:

```python
import numpy as np
data = np.load("results/sweep_Kh_N16.npz")
print(list(data.keys()))  # See available arrays
K_grid = data['K_grid']
```

---

## `results/n2_verification.npz`

N=2 VAN exactness verification across the (K, h) grid.

| Key | Shape | Dtype | Description |
|-----|-------|-------|-------------|
| `K_grid` | (20,) | float64 | Log-spaced K values |
| `h_grid` | (21,) | float64 | Linear h values |
| `exact_f` | (20, 21) | float64 | Exact dimensionless free energy $\beta f$ |
| `van_f` | (20, 21) | float64 | VAN dimensionless free energy $\beta f$ |
| `delta_f` | (20, 21) | float64 | Difference: VAN - exact |

**Expected:** `np.max(np.abs(data['delta_f']))` should be $< 10^{-4}$ (training limited, not model limited).

---

## `results/sweep_Kh_N{N}.npz`

Full (K, h) sweep results for system size N. Generated by `experiments/sweep_TH.py`.

### Grid arrays

| Key | Shape | Description |
|-----|-------|-------------|
| `K_grid` | (n_K,) | Dimensionless coupling grid (log-spaced) |
| `h_grid` | (n_h,) | Dimensionless field grid (linear, symmetric around 0) |

### Exact solution

| Key | Shape | Description |
|-----|-------|-------------|
| `exact_f` | (n_K, n_h) | Exact $\beta f = -(1/N)\ln Z$ |
| `exact_m` | (n_K, n_h) | Exact magnetization $\langle m \rangle$ |

### NMF solution

| Key | Shape | Description |
|-----|-------|-------------|
| `nmf_f` | (n_K, n_h) | NMF dimensionless free energy $\beta f$ |
| `nmf_m` | (n_K, n_h) | NMF magnetization |

### VAN with bias

| Key | Shape | Description |
|-----|-------|-------------|
| `van_bias_f_mean` | (n_K, n_h) | Mean $\beta f$ over seeds |
| `van_bias_f_std` | (n_K, n_h) | Std $\beta f$ over seeds |

### VAN without bias

| Key | Shape | Description |
|-----|-------|-------------|
| `van_nobias_f_mean` | (n_K, n_h) | Mean $\beta f$ over seeds |
| `van_nobias_f_std` | (n_K, n_h) | Std $\beta f$ over seeds |

### Error arrays (convenience)

| Key | Shape | Description |
|-----|-------|-------------|
| `delta_f_nmf` | (n_K, n_h) | NMF error: $\beta f_\text{NMF} - \beta f_\text{exact}$ |
| `delta_f_van_bias` | (n_K, n_h) | VAN (bias) error: mean $\beta f$ - exact |
| `delta_f_van_nobias` | (n_K, n_h) | VAN (no bias) error: mean $\beta f$ - exact |

### Array indexing

- First axis: K (coupling), index `i` → `K_grid[i]`
- Second axis: h (field), index `j` → `h_grid[j]`
- Example: `exact_f[i, j]` is the exact free energy at $(K, h) = $ (`K_grid[i]`, `h_grid[j]`)
- The h=0 column: `h0_idx = np.argmin(np.abs(h_grid))`

### Symmetry

Results for $h < 0$ are obtained by mirroring $h > 0$:
- Free energy (and errors): $f(K, -h) = f(K, h)$ (even in h)
- Magnetization: $m(K, -h) = -m(K, h)$ (odd in h)

---

## `results/finite_size.npz`

Finite-size scaling data at representative (K, h) points.

### Common arrays

| Key | Shape | Description |
|-----|-------|-------------|
| `system_sizes` | (n_sizes,) | System sizes tested, e.g. [2, 4, 8, 16, 32, 64, 128] |
| `representative_points` | (n_points, 2) | (K, h) pairs |

### Per-point arrays

For each representative point $(K, h)$, two arrays keyed as `K{K}_h{h}_delta_f_mean` and `K{K}_h{h}_delta_f_std`:

| Key pattern | Shape | Description |
|-------------|-------|-------------|
| `K{K}_h{h}_delta_f_mean` | (n_sizes,) | Mean $\Delta F/N$ at each system size |
| `K{K}_h{h}_delta_f_std` | (n_sizes,) | Std of $\Delta F/N$ over seeds |

Example keys: `K0.5_h0.0_delta_f_mean`, `K2.0_h0.5_delta_f_std`.

## Working With Results

### Quick inspection

```python
import numpy as np

data = np.load("results/sweep_Kh_N16.npz")

# What's available?
print(sorted(data.keys()))

# Grid info
print(f"K: {data['K_grid'].min():.2f} to {data['K_grid'].max():.2f}, {len(data['K_grid'])} points")
print(f"h: {data['h_grid'].min():.2f} to {data['h_grid'].max():.2f}, {len(data['h_grid'])} points")

# Worst-case VAN error
print(f"Max |delta_f| VAN (bias):   {np.max(np.abs(data['delta_f_van_bias'])):.6f}")
print(f"Max |delta_f| VAN (nobias): {np.max(np.abs(data['delta_f_van_nobias'])):.6f}")
print(f"Max |delta_f| NMF:          {np.max(np.abs(data['delta_f_nmf'])):.6f}")
```

### Extract a slice

```python
# Magnetization vs h at fixed K
K_target = 2.0
K_idx = np.argmin(np.abs(data['K_grid'] - K_target))
m_exact = data['exact_m'][K_idx, :]
h_vals = data['h_grid']
```

### Convert back to physical units

If needed, convert from dimensionless to physical:

```python
J = 1.0  # coupling constant
T = J / K  # temperature (for a given K)
H = h * T  # physical field (for a given h, at temperature T)
f_physical = T * beta_f  # physical free energy f = T * (beta*f)
```

Note that the $(K, h)$ parameterization mixes temperature into both axes, so a single point in $(K, h)$ space does not correspond to a unique $(T, H)$ pair — it corresponds to a **line** in $(T, H)$ space parameterized by $J$.
